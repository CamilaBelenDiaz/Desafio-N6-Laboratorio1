# Desafio-N6-Laboratorio1
Ejercicio práctico de Desarrollo Web: creación de index.html con título “Desafío 6”, aplicando etiquetas semánticas (enunciados, listas, strong, párrafos, imágenes, botones y contenedores). Se trabajan propiedades CSS de tipografía, color, fondo, width, height, padding y margin, integrando Google Fonts y transformando párrafos en elementos inline.
